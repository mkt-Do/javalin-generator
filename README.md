# Javalin Generator
Javalin Project Generator
If use this program, you must install `node` and `gradle`.
And clone this repository.
```
$ git clone ...
$ cd javalin-generator
$ yarn install
$ chmod +x index.js
```

## create project
```
$ ./index.js new hoge
...
$ cd hoge
$ gradle run
```
and access `localhost:7000`.
Display `hello world`.
